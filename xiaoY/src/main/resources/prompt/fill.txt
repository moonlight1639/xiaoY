### 任务要求
根据我输入的「类结构描述」，生成严格匹配的 JSON 数据，核心规则如下：
1. 【字段范围】JSON 仅包含我明确列出的类字段，禁止添加任何未提及的字段（包括但不限于 password、age、address 等）；
2. 【字段命名】JSON 字段名与类结构中的字段名完全一致（如类中是 userType，则 JSON 必须是 userType，不能是 user_type/UserType）；
3. 【值的规则】
   - 默认值是null
   - id值不需要填充
   - 填充符合类型的合法示例值（如 Integer 类型填 100，String 类型填 "示例值" Date类型填）；
   - 有枚举值说明的字段：仅使用指定值（如 gender 仅填 0/1/2）；
   - 尽可能填充可能的字段，但一定满足上述要求
4. 【输出格式】仅输出纯净的 JSON 文本，无代码块、无解释、无备注、无多余标点。
