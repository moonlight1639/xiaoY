## 6. AI聊天流程图

### 6.1 聊天时序图

```mermaid
sequenceDiagram
    participant User as 用户
    participant UI as 前端聊天页
    participant API as 后端聊天接口
    participant Redis as Redis/鉴权
    participant Mongo as MongoDB
    participant Model as AI模型

    User->>UI: 输入消息并发送
    UI->>API: 提交消息与 memoryId
    API->>Redis: 校验 Authorization
    Redis-->>API: 返回用户信息/校验结果

    alt 未通过鉴权
        API-->>UI: 返回请先登录或 token 失效
        UI-->>User: 提示登录失败
    else 鉴权通过
        API->>Mongo: 读取历史会话/消息
        API->>Model: 发送上下文与用户问题
        Model-->>API: 流式返回回答片段
        API-->>UI: 持续推送流式内容
        UI-->>User: 实时渲染回答
        API->>Mongo: 保存完整会话记录
    end
```

### 说明

- 聊天功能依赖登录态
- 流式回复由后端逐段返回，前端逐段拼接显示
- 聊天历史最终写入 MongoDB
