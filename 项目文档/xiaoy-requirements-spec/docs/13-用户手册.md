# 用户手册

## 1. 手册说明

本文档用于指导普通用户和管理员使用小Y校园智能生活服务平台。  
系统包含前台用户功能和后台管理功能，适用于演示、测试、培训和日常使用。

## 2. 系统简介

小Y是一个面向校园场景的智能服务平台，当前提供以下主要能力：

- 用户注册与登录
- 个人资料查看与修改
- 课程信息浏览
- 课程评价查看与提交
- 食堂与菜品信息查看
- 校车与校园生活信息浏览
- AI聊天问答
- 后台管理功能

## 3. 运行与访问说明

### 3.1 前端访问

启动前端服务后，通过浏览器访问系统首页。

### 3.2 登录说明

部分功能在未登录时也可浏览，但以下功能通常需要先登录：

- AI聊天发送消息
- 提交课程评论
- 修改个人信息
- 使用管理员后台功能

## 4. 普通用户使用说明

## 4.1 注册账号

### 操作步骤

1. 进入系统首页
2. 打开注册界面
3. 输入用户名、密码等注册信息
4. 点击注册按钮
5. 系统提示注册成功后，返回登录界面

### 注意事项

- 用户名应尽量唯一
- 密码不能为空
- 若注册失败，请检查输入是否完整

## 4.2 登录系统

### 操作步骤

1. 打开登录界面
2. 输入用户名和密码
3. 点击登录
4. 登录成功后，系统会保存当前登录状态

### 登录结果

- 登录成功：进入系统并保留登录态
- 登录失败：提示账号或密码错误，或其他错误信息

## 4.3 退出登录

### 操作步骤

1. 在系统页面中找到退出入口
2. 点击退出登录
3. 系统清除当前用户登录信息

### 退出后效果

- 当前账号的本地登录态会被清除
- 再次访问受保护功能时需要重新登录

## 4.4 查看首页

首页用于展示系统入口和主要功能导航。

### 可以完成的操作

- 浏览系统主要栏目
- 快速进入课程、食堂、评价、聊天等页面
- 了解平台整体功能布局

## 4.5 查看课程信息

### 操作步骤

1. 进入课程页面
2. 浏览课程列表
3. 查看课程名称、教师、简介等信息

### 说明

课程页面主要用于展示课程基础信息，帮助用户快速了解课程内容。

## 4.6 查看课程评价

### 操作步骤

1. 进入评价页面
2. 浏览课程评论列表
3. 如有详情入口，可进入评价详情页查看更多内容

### 可查看内容

- 评论用户
- 评论内容
- 时间信息
- 点赞/点踩等展示信息

## 4.7 提交课程评价

### 前提条件

用户需要先登录。

### 操作步骤

1. 进入评价或课程相关页面
2. 输入评论内容
3. 点击提交
4. 系统提示提交成功后刷新评论列表

### 常见问题

- 未登录：系统会提示先登录
- 内容为空：系统会提示内容不能为空或提交失败

## 4.8 查看食堂与菜品信息

### 操作步骤

1. 进入食堂页面
2. 浏览食堂地点列表
3. 查看对应菜品信息

### 可查看内容

- 食堂地点名称
- 菜品名称
- 菜品价格
- 菜品分类
- 菜品描述
- 菜品图片

## 4.9 查看校车信息

### 操作步骤

1. 进入校车页面
2. 查看校车相关内容

### 说明

当前页面主要用于展示校园校车相关信息，具体展示内容以系统页面为准。

## 4.10 查看校园生活信息

### 操作步骤

1. 进入生活服务页面
2. 浏览校园生活相关内容

### 说明

该页面主要用于提供校园生活服务类信息展示。

## 4.11 管理个人信息

### 前提条件

用户需要先登录。

### 操作步骤

1. 进入个人信息页面
2. 查看昵称、手机号、邮箱、头像等信息
3. 修改需要更新的字段
4. 点击保存

### 头像上传

1. 在个人信息页选择头像文件
2. 点击上传
3. 上传成功后刷新头像展示

### 注意事项

- 文件不能为空
- 如上传失败，请检查网络或文件格式

## 4.12 使用 AI 聊天功能

### 前提条件

用户需要先登录。

### 进入方式

进入聊天页面或 AI 测试页面。

### 操作步骤

1. 打开聊天页面
2. 若有历史会话，可在左侧选择会话
3. 若要开始新聊天，可点击新对话
4. 在输入框中输入问题
5. 按发送按钮或回车发送消息
6. 等待系统流式返回AI回答

### 聊天功能说明

- 支持查看历史会话
- 支持新建会话
- 支持实时显示 AI 回复
- 支持 Markdown 格式内容展示

### 常见问题

#### 发送消息没有反应

可能原因：

- 尚未登录
- 网络异常
- 后端服务未启动
- AI服务配置异常

#### 会话列表为空

可能原因：

- 当前账号尚无历史聊天记录
- 聊天数据尚未生成

## 6. 常见问题处理

## 6.1 无法登录

请检查：

1. 用户名和密码是否正确
2. 后端服务是否启动
3. 网络是否正常
4. Redis 登录态配置是否可用

## 6.2 页面加载失败

请检查：

1. 前端服务是否启动
2. 后端接口地址是否正确
3. 浏览器控制台是否报错
4. 后端日志是否存在异常

## 6.3 上传失败

请检查：

1. 文件是否为空
2. MinIO 是否正常运行
3. 后端上传接口是否可访问

## 6.4 聊天失败

请检查：

1. 是否已经登录
2. AI接口是否可访问
3. MongoDB 是否正常运行
4. 模型服务 Key 是否已正确配置

## 6.5 无法保存数据

请检查：

1. 输入内容是否完整
2. 后端数据库是否连接正常
3. 对应接口是否返回错误提示

## 7. 使用建议

1. 首次使用建议先注册普通账号进行体验
2. 使用管理员账号前先确认具备对应权限
3. 演示时建议按“登录 → 课程/食堂 → 聊天 → 后台管理”的顺序展示
4. 若用于答辩演示，建议提前准备测试数据与稳定网络环境

## 8. 联系维护说明

若系统在使用中出现持续性问题，建议维护人员重点检查：

- 前端服务运行状态
- 后端服务运行状态
- MySQL 连接状态
- Redis、MongoDB、MinIO 运行状态
- AI模型接口配置与可用性
